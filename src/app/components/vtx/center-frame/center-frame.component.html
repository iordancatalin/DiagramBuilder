This project aims to ease the work needed to create the tables of a database and the links between them. The application proposes a table creation system that any user, regardless of the level of knowledge about databases, can use and understand easily. Using this application, both the time and effort required to create a database are greatly reduced, giving the programmer time to focus on more important things.

The application allows creating multiple database charts and saving these charts for later reuse. Share with other people. Share your projects with others in a light and efficient way. The application offers the possibility of teamwork. Several people can work on the same chart at the same time. Changes to one of the participants can be seen by others without the need for the page to be reloaded.

Organize your tables in the work area as you wish. You can create relationships between tables, add or delete constraints. Modify the size of a table. You get to a much easier table by pressing the Ctrl + Shif + F combination you can look for a table by name. Generate SQl based on the diagram, only the diagram is a valid one.
