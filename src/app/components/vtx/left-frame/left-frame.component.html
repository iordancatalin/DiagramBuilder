Simple and fast

It is very easy and quick to create a database diagram, the interface being easy to understand and use. There is no need for advanced database knowledge. The interface uses a drag and drop system, the user can arrange the position of the tables using only the mouse. Additionally, the user can move the tables using the arrow keys on the keyboard.

The Ctrl + Shift + F key combination lets you search for a table in a database diagram

Exporta SQL

The application allows you to export a database diagram of a SQL database. Before export is executed in SQL format, a validation of the diagram is performed and is only exported if the database diagram is valid. The export can be done in several dialects of databases.

After the export is done, the user can copy the obtained script and create the database based on the diagram.

Share with others

If you do not want to work alone you can distribute your mucus with other people. The application allows the distribution of a diagram from almost any page, without having to navigate through several pages to make something so simple. When someone shares a diagram with you, you will receive a notification.

Real-time modifications

When working with multiple people on a diagram, you do not have to refresh the page to see the changes made by the other participants. Changes occur in real time, after the rescue is done, the data is refreshed providing great accuracy and consistency. Saving is done automatically after making changes.
